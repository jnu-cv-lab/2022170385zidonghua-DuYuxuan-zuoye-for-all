# 任务一实验报告：实现 Sinusoidal Position Encoding

## 一、实验目的

本任务的目的是实现 Transformer 中常用的 Sinusoidal Position Encoding。由于 Transformer 本身不像 RNN 那样按顺序处理序列，也不像 CNN 那样天然具有局部位置信息，因此需要额外加入位置编码，使模型能够区分不同 token 在序列中的先后位置。

## 二、实验原理

Sinusoidal Position Encoding 使用正弦函数和余弦函数生成固定的位置编码。其计算公式为：

PE(pos, 2i) = sin(pos / 10000^(2i / d_model))

PE(pos, 2i+1) = cos(pos / 10000^(2i / d_model))

其中，pos 表示序列中的位置，i 表示维度索引，d_model 表示模型维度。偶数维使用 sin 函数，奇数维使用 cos 函数。这样可以让不同位置对应不同的编码向量，并且不同维度具有不同的变化频率。

## 三、实验过程

本实验使用 Python 中的 numpy 实现位置编码矩阵的生成。实验中设置序列长度 seq_len 为 32，模型维度 d_model 为 16，因此最终生成的位置编码矩阵大小为 32×16。程序运行后，将完整的位置编码矩阵保存为 CSV 文件，并绘制了位置编码热力图以及前几个维度随位置变化的曲线图。

## 四、实验结果

程序运行成功，生成的位置编码矩阵形状为 (32, 16)。从 CSV 文件可以看到，第 0 个位置的编码结果为 0、1、0、1 交替出现，符合 sin(0)=0、cos(0)=1 的规律。热力图展示了不同位置和不同维度下编码值的变化情况，曲线图展示了前几个维度随位置变化的正弦和余弦波形。

## 五、实验结论

通过本任务，实现并验证了 Sinusoidal Position Encoding 的生成过程。该方法不需要训练参数，可以直接根据位置和维度计算得到位置编码。它能够为 Transformer 提供序列位置信息，使模型在处理序列数据时能够感知 token 的顺序和位置变化。
